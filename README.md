Sribe-App
A robust, full-stack blog platform built with python and Flask. This application demonstrates a deep implementation of relational database structures, secure user authentication, and dynamic content management.
Key Features:
1. User authentication
2. Relational Database Mapping
3. Administrative privileges

Tech stack:
1. Backend: Python, Flask
2. ORM: SQLAlchemy
3. Databse: SQlite
4. Forms: Flask-WTF, WTForms
5. Security: FLask-Login, Werkzeug
6. Frontend: Jinja2 templating, Bootstrap 5
